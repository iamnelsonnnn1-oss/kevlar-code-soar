# Ansible Training: Build a Small Lab (DevSecOps)

## 🎯 Project Overview
This repository is a hands-on lab designed to master **Ansible** for modern Software Engineering and DevSecOps workflows. We are building scalable, **Agentless Automation** in the **European Zone**.

## 🏗️ Architecture Design
- **Control Node:** Ansible Orchestrator.
- **Managed Nodes:** Ubuntu target servers via **SSH (Port 22)**.
- **Philosophy:** Scalability from 1 to 10,000+ hosts.

## 🛠️ Tooling
- Terraform (Provisioning)
- Ansible (Configuration & Hardening)
- Git (Version Control)
